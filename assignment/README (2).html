Portfolio Maker

The portfolio maker uses data from the AlphaVantage API.
Assumption is made that the user has basic knowledge about the stock market, ticker symbols and how stocks of a company work.

Features

Through choices from a menu provided,

  • A user can create multiple portfolios, each with multiple stocks and respective shares in them. User provides names for portfolios.
  • A user can create a flexible portfolio; buy stocks and sell a scecific number of shares for a stock on a specified date.
  • Provide a commission cost to the broker end for each transcation conducted in the portfolio.
  • Determine the total cost endured up until a certain date by portfolio. This includes the commission basis.
  • Compute the value of a portfolio on any specific date.
  • Fetches data from the AlphaVantage API for stock and shares data.
  • View the contents of any portfolio (stock name and shares).
  • Determine the total value or composition of a portfolio on any market-open day.
  • A user can create, store and retrieve any portfolio at any time. Persistence is achieved.
  • View all the pre-existing portfolios: names and pertaining data.
  • A user can view his custom portfolios by uploading a file path containing portfolio data. The file needs to be saved locally beforehand.
